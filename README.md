# 習慣トラッカー

毎日の習慣をチェックして記録・可視化するWebアプリです。

---

## できること

- **習慣の登録・管理**  
  続けたいことを登録して、絵文字やカテゴリで整理できます。

- **毎日のチェック**  
  今日やることが一覧で表示され、完了したらボタンを押すだけ。

- **連続達成日数（ストリーク）**  
  何日連続で達成しているかをリアルタイムで確認できます。

- **統計・グラフ**  
  週・月単位の達成率をグラフやカレンダーで振り返れます。

- **どこからでもアクセス**  
  スマホでもPCでも同じデータが見られます。

---

## 使い方

1. Googleアカウントでサインインする
2. 続けたい習慣を登録する
3. 毎日アプリを開いてチェックするだけ
